# gif-finder
A demo app featuring a bunch of Angular2 features, such as:
- Angular2 (beta 15)
- Typescript
- Material2 Design
- Component lifecycle
- Speech Recognition (HTML5)
- Http
- Observables
- Zone

# Run locally

You will need angular-cli to run this:

```
$ npm i -g angular-cli
$ npm i
$ ng serve
```

##### Enjoy
