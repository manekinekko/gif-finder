# gif-finder

A demo app featuring a bunch of Angular2 features, such as:
- Angular2 (@angular/*@0.0.0-3)
- Typescript
- Material2 Design
- Component lifecycle
- Speech Recognition (HTML5)
- Http
- Observables
- Zone
- SystemJS

# Run locally

You will need angular-cli to run this:

```
$ npm i -g angular-cli
$ npm i
$ ng serve
```
